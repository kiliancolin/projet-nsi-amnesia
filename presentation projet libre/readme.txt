Amnesia est un jeu d'horreur créé en 2010 par Frictional Games et son succès est en parti dû à la possibilité de moder (créer une histoire à partir des éléments que possède déjà le jeu) assez facilement.

Dans les fichiers de "redemption" (c'est le nom que j'ai donné à mon mod) il y a 4 choses : 
	-extra_english.lang correspond simplement à tout ce qui est texte dans le jeu.
	-custom_story_settings.cfg définit le nom du mod, qui est le créateur, quelle image voit on avant de lancer le mod, où sont rangé les "maps" (niveaux), dans quelle map commencer et à quelle endroit.
	-truc en plus, c'est juste un dossier pour moi, pour me souvenir de certaine choses, il sera supprimé quand j'aurai terminé.
	-maps, l'endroit où sont rangées toutes les maps et les fichier de script.

Vous ne pourrez pas ouvrir les fichiers .map car il faut avoir le jeu et l'éditeur de niveau donc je vous ai mit des captures d'écran de ce que j'ai commencé pour le projet libre.
Par contre, vous pourrez ouvrir les fichiers .hps, c'est là qu'est rangé tout le script. Le langage ressemble à du C mais le jeu possède aussi ses propres fonction.
Je vous met le lien du site où ils détaillent toutes les fonction présentes:
https://wiki.frictionalgames.com/page/HPL2/Engine_Scripts

J'ai choisi ce projet personnellement car je trouve le code de ce que j'ai déjà fait un peu "simpliste" et le fait de le faire comme devoir va me permettre de m'améliorer et d'en apprendre plus. Aussi j'aime tout simplement bien le jeu.
